Files to install a parallel version of Klipper mainline on Ender 5 Max.

The memos are here to explain the files compilation.

--------------------------------------------------------------------
Beta version, do not install it if you have no brain to fix/mod/custom.
---------------------------------------------------------------------
Install.sh is a fully automatic installer.
Install SSH command : wget --no-check-certificate https://raw.githubusercontent.com/christianKEL/E5M-CK/main/install.sh -O /tmp/install.sh && sh /tmp/install.sh
